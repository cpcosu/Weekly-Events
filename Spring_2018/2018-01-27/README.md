Weekly Challenge - Inverse Factorial
---

Given the value of `n!` (which can be very large), can you write a program to find the value of `n`?

Sample input:

```
10888869450418352160768000000
```

Sample output:

```
27
```

*Source: 2016 ICPC NAQ*
